Unit 01 - Introduction


Competencies

  1. A working knowledge of the Godot Engine and its core systems.
    1. Set up the Godot Engine and explore its interface.
    2. Understand the structure of game projects, i.e. assets, nodes, and scenes.
    3. Demonstrate the ability to run a simple project in Godot.

Activities

  1. Study or review the learning resources as needed.
  2. Install and configure the Godot Engine.
  3. Explore the Godot interface and basic project setup.
  4. Open and run a pre-made example project to understand its structure.
    • Pong (Godot 4.3)
  5. Create a Github repository for your game projects.

Deliverables

  1. Installed and configured Godot environment.
  2. Github repository for the next unit's project.
  3. A short video walkthrough explaining the Godot editor and project structure.
